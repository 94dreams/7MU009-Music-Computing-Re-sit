Synthesiser created by Ravinderpal Singh - 1214474

PLEASE ENSURE MIDI KEYBOARD INPUT CONNECTED BEFORE FOLLOWING INSTRUCTIONS 

1. Turn on DSP

2. There are three different waves patched in to the synthesiser, sine, saw and square.

3. Each wave is controlled by a slider, increase slider volume to hear more of each wave.

4. An ADSR unit is connected to control each wave, increase and decrease slider values to hear it work.

5. The delay feature is connected to the square wave, increase number values for left and right to hear.
